Projects

Project 1- MY CLASSMATES

The project is an interactive webpage that organizes names alphabetically with functional navigation and visually appealing design. It features grouped names with ordered alphabetic numbering, clickable links to Facebook profiles, and navigation icons (home, up, down, A-Z) for quick access. The layout is styled using internal stylesheets with enhanced spacing, letter-spacing, and background aesthetics, making it user-friendly and attractive.

Project 2- Quotations

The quotations.html file is a webpage showcasing at least 20 quotes, Bible verses, or pick-up lines, each with properly styled content and sources. The "Points to Ponder" section stands out with larger, emphasized text. The page features layered background colors and images, scrollable quote containers with margins, borders, and padding, and custom fonts for the first letters and sources of the quotes. A home button that will reload the page, providing navigation functionality, while internal CSS ensures a cohesive and visually appealing design.
